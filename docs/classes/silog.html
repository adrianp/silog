checkLevel

(
  • level
  • exists
)
Boolean private

Defined in src/silog.js:77

Checks if the given parameter is a valid silog logging level i.e. is an array of size 2, with the first element a number and the second one a string. Moreover, if the second parameter evaluates to true, the existence of the array in silog.LEVEL will be checked.

Parameters:

  • level Array

    the object to check.

  • exists Boolean

    if true, the existence of the object in silog.LEVEL will be checked.

Returns:

Boolean:

true if the object is a valid logging level, false otherwise.

LEVEL

Array final

Defined in src/silog.js:36

The logging levels used by silog; an order relation is established between them, with ASSERT (7) being the highest and VERBOSE (2) the lowest. If silog's level is set to X, any message with the level Y < X will not be logged.
